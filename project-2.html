Muzic – MBTI 기반 음악 커뮤니티 플랫폼

기간: 2025.10.20 ~ 2025.10.31

Java 21 Spring Boot Spring Security HTML CSS JavaScript Oracle Kakao Maps API

요약

- 관리자 승인 기반 음원 업로드로 신인·인디 뮤지션이 안전하게 작품을 공개할 수 있는 음악 커뮤니티
- MBTI 타입별 게시판 분리와 추천 로직을 통해 성향 기반 소통 경험 제공
- 출석 체크·룰렛·스토어 교환 등 이벤트로 참여 → 보상으로 이어지는 커뮤니티 루프 구현

역할

- 메인 홈 화면 및 마이페이지 UI/기능 구현(프로필/활동 내역 중심)
- 1일 1회 참여 가능한 룰렛 이벤트 구현 및 포인트 보상 흐름 연동
- GitHub 브랜치 분리 작업을 통해 팀 프로젝트 코드 관리 경험

성과

- 승인형 공개 + 이벤트 보상 구조로 콘텐츠 신뢰도와 서비스 참여도를 함께 강화
- MBTI 기반 커뮤니티 설계로 서비스 정체성과 차별점이 명확한 사용자 경험 구현
